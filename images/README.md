# 图片资源说明

本文件根据当前代码实际引用的图片文件名整理，按此命名并放置到`images/`及其子目录即可正常显示。

## 必须的图片（头像与首页）
- `首页-个人照片.jpg`：首页左侧的个人照片（建议尺寸：300x400，3:4比例）
- `顶部-头像.jpg`：多个页面顶部的圆形头像（建议尺寸：150x150）
- `avatar.jpg`：技能与兴趣等页面顶部头像（建议尺寸：150x150）

> 说明：原占位名`profile.jpg`、`detailed-photo.jpg`已不再在代码中使用。
