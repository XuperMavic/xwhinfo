# 个人信息展示网站

这是一个用于展示个人信息的静态网站模板，您可以通过GitHub或Gitee等平台托管，无需购买服务器。

## 网站特点

- **纯前端实现**：只包含HTML、CSS和JavaScript，无需后端代码
- **多页面设计**：包含首页、基本信息、专业履历、技能介绍、兴趣爱好和联系六个页面
- **密码保护**：专业履历和联系页面需要密码访问
- **响应式布局**：适配不同屏幕尺寸的设备
- **动态效果**：包含可收缩板块、轮播图、特殊图片动画等交互效果
- **详细注释**：代码中包含详细的注释，便于理解和修改

## 目录结构

```
Trae/
├── index.html       # 网站主入口文件
├── styles/          # 样式文件目录
│   └── main.css     # 主样式文件
├── scripts/         # 脚本文件目录
│   └── main.js      # 主脚本文件
├── images/          # 图片资源目录
│   └── README.md    # 图片资源说明
├── .gitignore       # Git忽略文件
└── README.md        # 项目说明文件
```

## 如何使用

1. **克隆或下载项目**：将本项目克隆或下载到您的本地计算机

2. **填写个人信息**：打开`index.html`文件，找到所有`[您的xxx]`或`[这里可以填写xxx]`的占位符，替换为您的实际信息

3. **准备图片资源**：按照`images/README.md`中的说明，准备所有需要的图片，并将它们放在`images`文件夹中

4. **自定义样式和内容**：根据您的需求，修改`styles/main.css`中的样式和`scripts/main.js`中的交互逻辑

5. **部署到托管平台**：
   - 如果使用GitHub Pages：将项目推送到GitHub仓库，然后在仓库设置中开启GitHub Pages
   - 如果使用Gitee Pages：将项目推送到Gitee仓库，然后在仓库设置中开启Gitee Pages
